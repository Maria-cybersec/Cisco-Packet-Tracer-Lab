# Cisco Packet Tracer Projects

## Overview
This repository contains 4 networking projects created while learning Cisco Packet Tracer.  
Each project was done to understand how networks work in real life using practical hands-on labs.

## Router and Switch

### What I did
I connected a router and a switch and tested communication between devices.

 Pinging was done successfully  
 Packet sending was successful  

### How I did it
I connected the router to the switch using Packet Tracer cables and assigned IP addresses to devices so they could communicate.

### Why I did it
I did this to understand how routers connect different networks and how data travels between devices.

## 4 Servers Network

### What I did
I built a network using:
 1 switch  
 3 PCs  
 4 servers (Google server, Yahoo server, DNS server, DHCP server)

 Pinging was successful  
 Websites opened from PCs  
 IP addresses were assigned automatically using DHCP  

FTP part was included but I forgot some configuration.

### How I did it
I connected all devices to a switch and configured servers for DNS, DHCP, and web services so that PCs could communicate and access websites.

### Why I did it
I did this to understand how real network services like DNS and DHCP work together in a real network environment.
